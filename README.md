# dotsama-rest-api
REST wrapper to polkadot.js API

# getting started

```bash
git clone https://github.com/metaspan/dotsama-rest-apidotsama-rest-api
cd dotsama-rest-api
npm install
# edit server.js, set PORT and upstream servers
node server.js
```
